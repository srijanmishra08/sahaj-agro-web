# Sahaj Agro Industries Website

A modern, high-performance B2B marketing website for Sahaj Agro Industries, a leading processor of premium Urad Dal based in Bhatapara, Chhattisgarh.

 This project replaces the legacy static site with a dynamic, visually rich experience designed to build trust with B2B buyers (wholesalers and retailers). It features smooth animations, a clear value proposition, and an interactive product catalogue.

### Key Features
*   **Premium Aesthetic**: Deep Green & Golden Yellow brand colors (`#113e2d`, `#FDB913`).
*   **Interactive Process Timeline**: Custom SVG-illustrated "Farm to Sack" process flow with hover animations.
*   **Product Catalogue**: Filterable grid displaying premium Urad Dal varieties (SQ, FAQ, Birds Eye).
*   **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
*   **Performance**: Fast-loading assets (WebP) and optimized CSS/JS.
